Project Overview

This capstone project is a 2D platformer built in Unity. It focuses on clean movement, readable visuals, and simple but satisfying gameplay loops. This project demonstrates my ability to design levels, implement core mechanics, and build a polished WebGL deployment, while keeping newcomers to games in mind.
